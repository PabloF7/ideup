### What is this repository for? ###

Basic micropython IDE for ESP8266 NodeMCU board

### How do I get set up? ###
Install Qt and Qt creator (online installed recommended).
Instal QScintilla.
Import with Qt Creator (New Project -> Import -> git clone) and compile.
